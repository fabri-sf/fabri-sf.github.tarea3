Diseño Web Responsivo

El diseño web responsivo es una técnica de desarrollo que permite que un sitio web se adapte automáticamente al tamaño y resolución de la pantalla en la que se está visualizando. Esto se logra mediante el uso de HTML, CSS y JavaScript.
